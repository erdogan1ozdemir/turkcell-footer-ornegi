# Turkcell Footer Örneği

`www.turkcell.com.tr` ana sayfasının **header + footer** birebir klonu üzerinde üç footer varyantı sunan tek dosyalık interaktif örnek. Sağdaki yüzen **V1 / V2 / V3** düğmeleriyle geçiş yapılır; masaüstü ve mobil responsive.

## Varyantlar
- **V1 - Mevcut:** canlı sitedeki footer'ın birebir kopyası (144 link, orijinal/yönlenen URL'ler, tekrarlar dahil).
- **V2 - Önerilen:** kanonik linkler, tekrar temizliği, iPhone politikası (15+ ve 13 128 GB), yeni yüksek-trafik sayfaları (Hız Testi, SuperBox, eSIM, Müşteri Hizmetleri, Kurumsal/B2B - sarı nokta ile), KVKK/Çerez/Kullanım Koşulları/Mesafeli Satış/Site Haritası yasal barı, App Store/Google Play rozetleri. Mobilde başlık altında dengeli 2 kolon.
- **V3 - Konsept:** V2 ile aynı detaylı sütunlar + üstte hızlı-işlem karoları (Fatura Öde, TL Yükle, Hız Testi, Sipariş Takibi, En Yakın Mağaza, Destek) + altta Turkcell Uygulaması bloğu.

## Kullanım
`index.html` dosyasını tarayıcıda aç. Harici bağımlılık yok; görseller ve font Turkcell CDN'inden çekilir.

> Yalnızca tasarım/İA örneği amaçlıdır; Turkcell ile resmi bağlantısı yoktur.
